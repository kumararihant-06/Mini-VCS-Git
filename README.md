# Mini-VCS

A simple version control system implemented in C++ with storage deduplication.

## Build
```bash
make
```

## Commands (Coming Soon)
- `./minivcs init`
- `./minivcs add <file>`
- `./minivcs commit -m "message"`
- `./minivcs log`
- `./minivcs checkout <id>`
